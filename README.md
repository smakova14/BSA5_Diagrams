# BSA5_Diagrams

Проект по построению диаграмм: DFD, Swimlane, State Diagram и State Table для барбершопа (BRB) и службы доставки (DLV).

## 🛠 Навыки
- Data Flow Diagram (DFD)
- Swimlane Diagram
- State Diagram
- State Table
- Business Process Modeling

## 📋 Описание проекта
В ходе работы я построила несколько типов диаграмм для визуализации требований к системе онлайн-записи барбершопа и системы управления доставкой.

### Что именно было сделано:
1. **DFD уровня 0 (BRB)**: Построила диаграмму потоков данных с внешними сущностями, процессами и хранилищами.
2. **DFD уровня 0 (DLV)**: Построила диаграмму потоков данных для службы доставки.
3. **Swimlane (BRB)**: Построила диаграмму бизнес-процесса записи клиента.
4. **Swimlane (DLV)**: Построила диаграмму бизнес-процесса обработки заказа.
5. **Диаграмма состояний (BRB)**: Построила жизненный цикл учётной записи Мастера.
6. **Диаграмма состояний (DLV)**: Построила жизненный цикл Заказа.
7. **Таблица состояний (BRB)**: Составила таблицу переходов для сущности «Услуга».
8. **Таблица состояний (DLV)**: Составила таблицу переходов для сущности «Курьер».
